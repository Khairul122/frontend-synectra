# Synectra Frontend

Frontend application for the Synectra project, a platform for client reception and management.
Built with React, Vite, Tailwind CSS, GSAP, and Barba.js, featuring a modern Neubrutalism design system.

## 🚀 Fitur yang Sudah Dibuat (Implemented Features)

- **Authentication**: Halaman Login & Registrasi.
- **Dashboard**: Role-based view untuk admin dan client.
- **Portfolio Management**: Menampilkan dan mengelola data portofolio.
- **Order Management**: Pelacakan status pesanan dan laporan progres.
- **UI & Animasi**: Visual 3D interaktif, GSAP animations, dan page transition yang mulus.
